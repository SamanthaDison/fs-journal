**2.** What is the `Singleton` design pattern?
<!-- enter you answer in the space below -->
```
A design pattern that instantiates a class to a single instance. We use this when creating an export const for services and instantiating a single const for the entire service class.
```
**3.** What is the `Observer` design pattern?
<!-- enter you answer in the space below -->
```
A design pattern in which an object maintains a list of observers or listeners that track and notify any changes within the state. Examples include ProxyState.on in the MVC pattern and computed variables in Vue.
```
**4.** What is the `Strategy` design pattern?
<!-- enter you answer in the space below -->
```
A design pattern that encapsulates like methods and extends upon them. I.e., defining a family of algorithms and putting each in a separate class.
```
**5.** What is the `Factory` design pattern?
<!-- enter you answer in the space below -->
```
A design pattern that define an interface or abstract class for creating an object but let the subclasses decide which class to instantiate.
```
**6.** What is test driven development?
<!-- enter you answer in the space below -->
```
Test driven development is a development method that requires subjecting the code to tests throughout the development process as opposed to testing the code after dvelopment is finished.
```
