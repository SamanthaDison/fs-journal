# CSS

Take advantage of display box in dev tools to check margins and displays
    Margin: 0 to remove margins
    Margin: top, right, bottom, left (if you don't specify margin, enter values in clockwise direction)
    Sibling/same container: padding; Different items/container: margin
    em descriptor will move margins based on size of parent text

Position absolute VS position relative:
    PR - moves indv. elements relative to its current position
    PA - moves elements out of the DOM flow and positions elements relative to the parent 

Use notes/regions to separate ideas 
Use root selector to set color schemes and easily change multiple variables

Div and section tags will block out display to the full (or set) width; flexbox allows us to wrap text   
Flexbox:
    Section out and make mutliple columns/sections for each display flex
    Justify: horizontal; Allign: vertical
    Bootstrap syntax start + end; CSS syntax left + right
