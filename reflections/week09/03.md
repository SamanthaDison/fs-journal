## Questions

1. What is a Base64 Encoded Image?
A Base64 image is an image that has been encoded from binary into a text format.

2. What are the drawbacks of using Base64 Encoding?
Base64 encoding increases the file size by 25% leading to further strain on bandwith. Images are also unable to store locally so must be decoded and rendered upon every pageload. They are also unaccessible via public URL which can make accessing your webpage via image harder.

3. Why are these drawbacks issues in this day and age versus in the past?
CPU overhead, caching, and user-interface. Browsers have become much more effiecent at rendering and retrieving data so using Base64 encoding can actually work against modern/optimized HTTP connections.
