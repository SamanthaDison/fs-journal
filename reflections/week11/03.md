## Questions

1. What is SQL injection?
SQL injection is where someone will use and write SQL code within a query (think like a form submission).

2. What are 3 methods SQL injection can be done by?
Form injections (database queries), modify cookies, forged HTTP headers

3. How can we detect and sanitize SQL injection attacks?
Firewalls and intrusion detection systems will help to detect SQL injection attacks. We can prevent them by sanitzing our code with readonlies and other safeguards such as 'limit 1' with delete queries.
