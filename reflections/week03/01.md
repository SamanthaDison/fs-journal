## Questions

1. What problem does using exports solve?
Using exports solves ahving mutliple script tags in HTML thus reducing loading speed and increasing functionality. It also helps to solve naming conflicts.

2. How does export differ from export default?
Exports can be used for multiple modules, multiples times. Default exports can only be used one per module and exporting single values.

3. What is a benefit of using the Module System?
There are multiple benefits to using the Module System such as: increased functionality, code can be split into smaller files improving legibility and open-closed principles, there is no need to use strict mode as the use of modules already 'hides' the code from the user.
