## Questions

1. What is the purpose of Encapsulation?
Prevents bugs from using a shared mutable space as well as makes troubleshooting and adjusting code easier.
2. What were some of the problems with closures and the underscore prefix?
Underscore prefixes are only used internally, so if if their implementation changes, this could lead to the code breaking.

3. How do we create private variables in a ES6 Class? Why would you do this?
By using a privileged method; they do not rely on convention end enforce true encapsulation.
