## Questions

1. What are the three categories of data types? How are they different?
Value Type: variables directly contain data
Reference Type: does not store data directly; references or points to another memory location that stores the data: Strings, Arrays, Classes, Delegates
Pointer Type: does not store data but reference or points to another variable or memory location; declared using *
