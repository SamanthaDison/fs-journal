## Questions

1. What is an Enum, and what are some use cases for them?
An enum is a constant type for storing names or strings to int values. Think of enums almost as a 'list' of a type where each value has a corresponding int value. They are great for code maintainence allowing you to use int values by assigning a meaningful name.

2. How can you modify an Enum?
You may assign int values to your enum members and may cast strings to ints to store in an enum.

3. How have you used Enums in your afternoon lab projects this far?(if you have not yet, give an example of how you could)
Enums may be used to store to store outcomes such as our rock, paper, scissors game. You may store the various choices/outcomes in an enum and cast the user input from the console to iterate over the enum and compare values to return win or loss.
