## Questions

1. What is a List in C#?
Lists are essentially a collection of reference types much like how we use arrays to store objects in JavaScript.

2. What List methods seem like you might use them often? Why?
List.Add for adding data into the database
List.Contains for filtering data
List.Find and List.FindIndex for returning reference types
List.Remove for removing data from databse
List.Sort for sorting/filtering

3. How would you retrieve an item from a list? What method could you use?
List.Contains, List.Find, List.FindIndex
