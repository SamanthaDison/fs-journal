## Questions

1. What is the purpose of `Async`/`Await`?
Async/Await allows for further abstraction and simplification of promises/callbacks. Code looks sychronous but is running asynchronous.

2. What must you do in order to  `await` a promise inside of a function?
The client function must be called as async.

3. What are some of the primary benefits of `Async`/`Await`?
These functions can be chained easily and the syntax is more readable. Also looks like and can be debugged like synchronous code.
