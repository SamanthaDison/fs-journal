## Questions

1. What is Code Documentation? What are some of the ways to document code?
Code documentation is a method of leaving notes or 'documenting' your code so that it is clear and easy to understand by other developers. Some ways to document code are by detailing what functions do, parameter types/expectations.

2. What are some benefits of Code Documentation?
Code documentation helps other developers understand your code so they can easily add to or adjust it. It is also handy for ourselves as it can allow us to easily identify certain functions and provide clarification in case we have not worked on that piece of code in a while.

3. How important is Code documentation and why?
Documentation is especially important now as 'coding' has become a collaborative effort; it will save both you and your colleagues time and headaches down the road.
