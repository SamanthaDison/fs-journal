## Questions

1. What is a virtual property?
Virtual properties are properties that do not persist in the database, they are additional fields built into model that are usually accessed with certain functionality such as .populate()

2. When might you use a virtual property? 
Virtual properties are commonly used to populate or access user information. 

3. How do you search by a virtual properties value?
Virtual properties are not searchable by query or field selections; in code they are accessed by set or get functions.
