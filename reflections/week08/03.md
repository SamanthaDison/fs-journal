## Questions

1. What is Software Development Testing?
Development testing involves unit testing or testing 'units' of code during the process of development. Involves testing as you go rather than testing a larger system down the road.

2. What are the benefits of Testing in software development?
Unit testing allows for ensuring smaller components of the code functions as needed. This helps in ensuring the code as a whole will be more likely to function down the line. Development testing will also help catch bugs before the product is passed to the QA team.

3. What are some potential drawbacks of Testing in software development?
Testing does not always account for all the ways code may fail. Testing individual components can also be time-consuming if one isn't already in the habit of doing so; unit testing will ultimately save time down the road though.
