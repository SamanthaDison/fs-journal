## Questions

1. What is a Pseudo-Class and what are some of the most common ones you think you will use.
A pseudo-class is a keyword added to a selector to that specifies a special state of the specified element. Some of the most common ones I think we will use are :hover, :root, :only-child, :first-child, :after, and :before.

2. What is Specificity and how might you use it to your benefit?
Specificity is a trait of CSS that indicates which rule shall be expressed when multiple rules target the same element. One can utilize specifity to organize their code and keep it neat. Specificity is used to target specifc elements and determine what we want expressed in our code.

3. What problems do you think you could run into if you over-utilized the !important feature?
The !important feature overrules specificity, over-using it can cause inconsistencies in the code and load rule. It can also lead to our code being busier than it needs to be.
