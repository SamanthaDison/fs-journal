## Questions

1. In your own words, why do we use Git?
We use Git to allow other developers and potential employers to easily view and access our code.

2. What are the benefits of having multiple branches?
Multiple branches allow developers to create parallel developments of the same code. 

3. What is the difference between Git and GitHub
Git is software used for storing code locally; GitHub is a cloud-based server for hosting code.
