## Questions

1. What is Scope ?
Scope defines where variables are available for use.

2. What is Hoisting ?
A JavaScript mechanism where variables and functions are moved to the top of their scope.

3. In what cases might you use let vs const vs var?
Const is used to define variables that cannot be modified. Let is best default for defining variables but are block scoped so will only apply to the scope of a specific function. Var is globally scoped and therefore may not manipulate quite in the way we want them to within a function.
