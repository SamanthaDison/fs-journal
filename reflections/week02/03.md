## Questions

1. What are the main ways to write information to the console? Why/when should you use each style.
Console.log() You can utilize console.log() to test and make sure functions are working.

2. Which tab allows you to see the breakdown of HTML/CSS and how can this tab be useful when debugging HTML?
The Source Tab; this is useful for debugging as it will run through the code step by step and will actuall pause the code on the line where a function runs. There is also the Code Editor feature that will allow you to alter code in real time.

3. Outside of writing everything to the console, what is a better way to debug your code?
Using a debugger and taking advantage of Break Points to better specify and identify specific functions.
