## Questions

1. What are the three ways to syntactically write a function? What are the differences in how the function acts (if any)?
Function declarations: function definition is hoisted, allowing the function to be used before it is defined.
 Function expressions: an unnamed function that is not hoisted and cannot be used before being defined.
Arrow function expressions: a shorter way to write function expressions.

2. What is the difference between Parameters and Arguments?
Parameters define functions and are passed through the function. Arguments are the conditions of the function or what is the function is doing.

3. What are higher order functions? Can you provide an example?
High order functions are functions that accept another function as a parameter; an example would be array.prototype.filter
